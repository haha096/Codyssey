pickle 모듈의 사용
: 파이썬에서 복잡한 데이터(딕셔너리, 리스트 등)을
  원래 모습 그대로 보관하는 모듈

pickcle.dump()와 pickcle.load()의 사용
: 복잡한 데이터를 얼리고 그 얼린 데이터를 해동하는 함수

-> 즉, 이진파일을 읽어올 때는(write() 사용) 글자가 아니라서 에러가 발생
   pickcle.dump()를 사용해 파일을 얼려서 저장

   파일을 가져올 때는 read()로 사용하면 외계어 상태로 전달됨
   pickle.load()를 사용해 얼려진 데이터를 복원


텍스트 파일과 이진파일의 차이점과 장단점
텍스트 파일
-> 사람이 읽을 수 있는 파일

- 호환성이 좋음
  : 사람이 읽을 수 있는 파일이다보니 엑셀, 메모장 등의 프로그램에서
    쉽게 열고 읽을 수 있습니다

이진 파일
-> 컴퓨터가 읽을 수 있는 파일

- 텍스트 파일보다 빠른 속도
  : 컴퓨터 메모리 상태에서 읽을 수 있는 파일이기 때문에
    읽고 쓰는 속도가 텍스트 파일보다 훨씬 빠릅니다

- 호환성이 떨어짐
  : 이진파일을 읽어올 때는 메모장, 엑셀과 같은 프로그램과
    호환성이 좋지 않습니다
