[주요 메소드 상세 설명]
def __init__
-> 계산기의 연산 흐름을 제어하는 5가지 내부 상태(입력값, 이전값, 연산자, 상태 플래그 등)를 초기화함
-> 입력 단계에 따라 동작이 변하는 상태 머신(State Machine) 구조를 설계하여 로직의 일관성을 확보함

def set_operator
-> 사칙연산 버튼 클릭 시 연산자를 등록하고 연속 연산의 흐름을 제어함
-> 이전 연산이 완료되지 않은 상태에서 새 연산자가 들어올 경우, equal()을 선행 호출하는 로직을 통해 중간 결괏값을 자동 산출하도록 알고리즘을 구성함

def _validate_result
-> 연산 결과의 수학적 무결성을 검증하고 오버플로우 및 NaN 여부를 판단함
-> 외부 라이브러리 없이 **IEEE 754 부동소수점 표준의 특성(value != value)**을 활용하여 NaN을 감지하고, 무한대(inf) 비교를 통해 수치 예외를 차단함

def _format_number
-> 계산된 실수를 사용자에게 보여주기 적합한 문자열 형태로 정제함
-> round() 함수와 문자열 rstrip() 알고리즘을 결합하여, 소수점 6자리 반올림 및 정수형 출력 시 불필요한 소수점(.0)을 제거하는 최적화된 포맷팅을 구현함

def _update_display_font
-> 결과값의 길이에 따라 디스플레이 폰트 크기를 실시간으로 조절함
-> 문자열 길이를 입력값으로 받아 역비례 구간별 폰트 매핑 알고리즘을 적용하여, 긴 숫자 입력 시에도 UI 레이아웃이 깨지지 않도록 동적 스케일링을 처리함

[전체 코드 흐름]

1. UI 초기화 및 바인딩: PyQt5를 이용해 화면을 구성하고, 각 버튼에 람다(Lambda) 패턴을 적용하여 Calculator 로직과 연결함
2. 데이터 입력 단계: 숫자나 소수점 입력 시, _new_number 플래그를 확인하여 새로운 숫자를 시작할지 혹은 기존 문자열에 이어 붙일지(Concatenation) 결정함
3. 연산 대기 및 처리: 연산자 버튼 클릭 시 현재 입력된 값을 _previous_input으로 넘기고, 연산자를 대기 상태로 등록함 (연속 연산 시 중간 계산 수행)
4. 최종 연산 실행: = 버튼 입력 시 저장된 연산자와 피연산자를 바탕으로 사칙연산 수행 후, **중앙 집중식 예외 처리(try-except)**를 통해 계산의 안정성을 검증함
5. 출력 및 피드백: 연산 결과에 대해 반올림 및 포맷팅을 거친 후 UI에 전달하며, 텍스트 길이에 맞춰 폰트 크기를 조절하여 화면을 갱신함
